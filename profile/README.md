## Что это
Кроссплатформенная утилита для подключения, прошивки и полной настройки контроллеров полёта с прошивкой Betaflight. Поддерживает тюнинг PID, фильтров, профилей и телеметрии.

## Основные возможности
- Прошивка контроллера (Firmware Flasher), выбор target/версии  
- Вкладки Setup/Ports/Configuration/PID Tuning/Modes/OSD  
- Экспорт/импорт конфигов, резервные копии  
- CLI для тонких правок, работа с логами Blackbox

## Расширенные функции
- Онлайн/офлайн пакеты прошивок  
- Настройка RX/телеметрии/VTX/LED  
- Профили под разные типы рам и моторов

## Как начать
1) Загрузите и установите приложение.  
2) Подключите контроллер по USB.  
3) Нажмите Connect и считайте текущие параметры.  
4) При необходимости прошейте, затем настройте и сохраните профиль.

## Системные требования
Windows 10/11 x64, 2-ядерный CPU, 4 ГБ RAM, 300 МБ диска, USB-кабель.
